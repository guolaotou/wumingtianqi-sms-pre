# For测试，需要加快系统时间
 
## 脚本地址
> scripts/speed_macbook_time.go

## 脚本功能
设定参数，int值，表示以n秒代替一天，然后循环运行。
脚本起始时间为系统当前时间

## 注意
由于是sudo命令，所以需要用sudo执行脚本
